A Resource is equivalent to a blog post. Resources are always "owned", by one owner. An owner can be a user or an organization.

- `authors` - A list of people or entities who were responsible for creating the resource, with their roles.
  - Invite existing users on the platform to be authors on the Resource, whereby the resource is listed on their page, and the page is linked on the resource?
- `banner` - media to place as a background in the hero section (image or video)
- `externalResources` - links to external websites where you can find the same resource. Useful for landing pages
- `id` - A unique identifier for the resource
- `image` - primary image
- `owner` - The canonical owner of the resource
- `summary` - A short description or tagline for the resource. Used for SEO.
- `tags` - arbitrary tags that can be used in site-wide search & filtering
- `title` - The title for the Resource
